# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Space-bunny18/ElevateCV.git
```

```bash
cd ElevateCV
```

---

## Install Backend Dependencies

```bash
cd backend
```

```bash
pip install -r requirements.txt
```

---

## Start Backend

```bash
uvicorn main:app --reload
```

Backend runs on:

```
http://127.0.0.1:8000
```

---

## Open Frontend

Open

```
index.html
```

using Live Server or any local web server.
